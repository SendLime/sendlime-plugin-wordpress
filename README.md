# SendLime Plugin for WordPress

This is the WordPress plugin for SendLime. This plugin lets you notify your customers about their WooCommerce order updates via SMS. To use it you will need a SendLime account. Sign up for free at [sendlime.com](https://www.sendlime.com).

For full API documentation refer to [developer.sendlime.com](https://developer.sendlime.com) or the live API reference at [api.sendlime.com/api-docs](https://api.sendlime.com/api-docs).

This version uses the current SendLime API v2:

- Base URL: `https://api.sendlime.com/api/v2`
- Authentication: `Authorization: Bearer sl_live_...`
- SMS endpoint: `POST /messages`

Create an API key from the SendLime dashboard and paste that key into the plugin settings. API secrets are no longer required.
